## JIRA 이슈명 규칙

- **[주제-개발단위] 날짜 작업내용**

  - 대괄호 뒤에 띄어쓰기
  - 날짜 뒤에 띄어쓰기

  ex) [BE-회원] 0720 로그인 기능 개발

  [FE-치료] 0720 치료 입력 기능 개발

  [UI-RTC] 0720 화상 코칭 UI 디자인

  [Docs] 0720 API 설계 보완

- 주제

  - BE, FE, UI, Infra, Docs, Lec, Plan ...

- 개발단위
  - 회원, 치료, 처방, RTC, ...

**story** - 개발 관련 (코드를 썼냐 안썼냐로 구분)

**task** - 개발 관련 x
